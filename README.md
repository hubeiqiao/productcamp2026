# How a Product Person Ships Real Products

ProductCamp 2026 Ottawa presentation by Joe (Beiqiao) Hu.

**Live:** [story.joespeaking.com](https://story.joespeaking.com)

## About

A 25-slide interactive presentation about building [Joe Speaking](https://joespeaking.com) as a solo product person using Claude Code and Codex. Covers the full journey: product decisions, AI workflow, debugging, validation, and what it actually takes to ship a real product.

## Tech

- Single `index.html` with embedded CSS and JS
- ASCII particle title animation (canvas-based, spring physics)
- Keyboard/touch/swipe navigation
- Progressive reveal on survey slide
- Embedded video with custom controls
- Live AI usage dashboard via iframe embed
- Hosted on Cloudflare Workers at [story.joespeaking.com](https://story.joespeaking.com)
